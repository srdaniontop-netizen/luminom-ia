# ⚡ Quick Start - Luminom IA

¿Quieres probar Luminom IA rápidamente? Sigue estos pasos:

---

## 📋 Pre-requisitos

- ✅ Node.js instalado
- ✅ MongoDB (local o Atlas)
- ✅ API Key de Claude (Anthropic)

---

## 🚀 Inicio Rápido (5 minutos)

### 1. Clonar e Instalar

```bash
git clone https://github.com/tu-usuario/luminam-ia.git
cd luminam-ia/backend
npm install
```

### 2. Configurar

```bash
cp .env.example .env
```

Edita `.env` y configura al menos:

```env
MONGODB_URI=mongodb://localhost:27017/luminom-ia
JWT_SECRET=tu_secreto_aqui
ANTHROPIC_API_KEY=sk-ant-api03-tu-clave-aqui
```

### 3. Ejecutar

```bash
npm run dev
```

### 4. Usar

1. Abre http://localhost:5000
2. Regístrate
3. ¡Empieza a chatear con tu tutor IA!
